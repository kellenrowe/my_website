Warbler

Warbler is a Twitter clone built with Python, Flask, PostgreSQL, and SQLAlchemy, Jinja templating, and jQuery. This was already a partially functioning app, my primary roles were to find bugs, fix them, then test my solutions. I accomplished this by going through each route and test, one by one, to find discrepencies. After this process was complete, I also added more tests, to the models and views. This was one of the earlier large codebases I'd worked on. As it was written primarily by others, it was a great introduction into the real world task of getting up to speed with your new team.

Go to Warbler!

Jobly

Jobly is a full-stack application that matches job seekers and employers. Jobly was built using a Node / Express backend and a React frontend. The application allows logged in users to browse and search hiring companies and jobs by name or company. It also allows users to view / edit their profile and apply for specific jobs.

I used test driven development (TDD) on the backend and practiced thinking through a detailed component hierarchy on the frontend.

Go to Jobly!

Pixly

Pixly is a full-stack application that allows users to upload and edit their photos. I used Python and Flask on the backend, rendering HTML with Jinja for the frontend. Users can also search existing pictures by Photgrapher or Caption.

This project allowed me to think critically, not about which technologies I wanted to use, but instead, about those which optimized the engineering process.

Go to Pixly!

Covidnames

Covidnames is my take on the popular boardgame Codenames. It's a fulltack app that uses the WebSocket protocol with Socket io. Anyone with an internet connection can join the same board and enjoy a game. Players can generate words automatically or choose three topics which the app will use to fetch related words from an API.

This project solidified best practices when working with AJAX. It also taught me one way of working with the WebSocket Protocol.

Go to Covidnames!
